        Name: FMPExample
      Status: Windows
       Level: Beginner
  Maintainer: Erik Ylvisaker   ylvisaker@ucdavis.edu
     Summary: Loads and displays FMP maps.  Uses the CL_FlexibleMap contrib class.

This program loads and displays some example FMP maps.  Arrow keys allow you to scroll around the map, and numeric keypad + and - allow you to cycle through the maps. 

FMP maps are produced by Mappy, you can download the latest version of Mappy at http://www.tilemap.co.uk/

Included maps are the example maps that come with Mappy.
