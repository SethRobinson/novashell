         Name: PilotsPuzzle
       Status: Windows(Y), Linux(Y)
        Level: Intermediate
   Maintainer: Core developers
      Summary: A puzzle game

Demonstrates a small puzzle game, using display, gui, input, etc.

To solve puzzle: You should make all items horizontal.
When you click at item, all items on that row and column are flipped.

You can see hints by clicking on: Show hints On/Off button
You can start new game by clicking on: New Game button
